P25Scanner Installation Instructions
=================================

1. Prerequisites:
   - Windows 10 or later
   - .NET 8.0 Runtime (or SDK for development)
   - RTL-SDR device

2. Installation Steps:
   a. Extract all files from this ZIP archive
   b. Navigate to the extracted folder
   c. Double-click P25Scanner.exe to run the application

3. First Time Setup:
   - Connect your RTL-SDR device before launching the application
   - The application will automatically detect your RTL-SDR device
   - Use the interface to configure your scanning preferences

4. Troubleshooting:
   - Ensure your RTL-SDR device is properly connected
   - Make sure .NET 8.0 Runtime is installed
   - Check Windows Device Manager to verify RTL-SDR driver installation

For support or issues, please visit the project repository.
